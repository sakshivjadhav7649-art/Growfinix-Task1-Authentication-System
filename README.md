# Secure Authentication System

## Technologies Used

- Node.js
- Express.js
- PostgreSQL
- JWT
- bcrypt

## Features

- User Registration
- User Login
- Password Hashing
- JWT Authentication
- Protected Routes

## API Endpoints

### Register User
POST /api/auth/register

### Login User
POST /api/auth/login

### Protected Route
GET /api/auth/profile

## Setup

1. Install dependencies

npm install

2. Start server

node server.js
